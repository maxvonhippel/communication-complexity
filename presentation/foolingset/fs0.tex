Consider a two-party protocol for determining whether two inputs are equal:
\pause
\begin{example}[$Equality$]
\(EQ : \mathbb{B}^n \times \mathbb{B}^n \to \mathbb{B}\)
is precisely
\(\langle x, y \rangle \mapsto 1\)
if $x = y$ else 0.
\end{example}
\pause
Example protocol $\Pi$:\\
\begin{leftbubbles}
\texttt{P1}$=x$.
\end{leftbubbles}
\pause
\begin{rightbubbles}
\texttt{P2}$=1 \text{ if } y = x \text{ else } 0$.
